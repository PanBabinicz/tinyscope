# tinyscope

## Introduction

> The **tinyscope** is a minimalist, open-source oscilloscope project built around the CH32V003 RISC-V
> microcontroller and a small OLED display.
>
> The idea behind tinyscope is simple:
>
> Build the smallest, most affordable, and most accessible oscilloscope possible — without sacrificing educational
> value. The tinyscope explores how much performance and functionality can be extracted from a tiny
> microcontroller with limited memory and resources. By leveraging the built-in ADC of the CH32V003 and driving a
> compact OLED screen, the system can sample analog signals and display real-time waveforms in a pocket-sized
> form factor.
>
> This project aims to:

- Encourage learning in embedded systems and signal processing
- Demonstrate efficient use of constrained hardware
- Provide a low-cost diagnostic tool for hobbyists and students
- Showcase what is possible with ultra-low-cost RISC-V microcontrollers

> The tinyscope embraces hardware simplicity, firmware efficiency, and clean system design. Every byte of memory
> and every CPU cycle matters — making it both a practical tool and a technical challenge.
